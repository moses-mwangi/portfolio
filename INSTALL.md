# oh-my-opencode Installation

```bash
curl -s https://raw.githubusercontent.com/code-yeongyu/oh-my-opencode/refs/heads/master/docs/guide/installation.md | bash
```

Or globally install the CLI:

```bash
npm install -g oh-my-opencode
```

For full installation instructions, visit: https://raw.githubusercontent.com/code-yeongyu/oh-my-opencode/refs/heads/master/docs/guide/installation.md
